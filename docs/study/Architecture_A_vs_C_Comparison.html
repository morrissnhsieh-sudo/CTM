Verdict: Architecture C — Greenfield Microservices

Arch C scores 37/40 vs Arch A's 29/40. It wins on real-time sync (Yjs CRDTs), AI-nativeness (MCP server + LangGraph + pgvector), scalability (independent K8s services), and license safety (all MIT). Arch A's only advantage is a faster MVP (3 mo vs 5–6 mo) — use Arch A only if you need to ship in <3 months, then migrate to C.
